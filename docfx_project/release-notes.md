# Release Notes

## Version 9.0.2

Current release with full support for .NET 10.0.

### Features
- Multi-language HTTP client proxy generation (C#, TypeScript, Python)
- DTO generation with proper type mapping across languages
- Roslyn-based code analysis for ASP.NET Core controllers
- Command-line tool for automated code generation
- Support for generics, nullable types, and inheritance hierarchies

### Target Frameworks
- .NET 10.0 (primary)
- .NET Standard 2.0 (core libraries)

## Version 9.0.x

### Changes
- Upgraded target framework to .NET 10.0
- Updated Microsoft.CodeAnalysis dependencies
- Enhanced C# file declaration with indexing support
- Fixed member access expression generation

## Version 8.x

### Changes
- Added Python async HTTP client generation
- Improved TypeScript type mapping
- Enhanced generic type support
- Various bug fixes and performance improvements
